# DFS & BFS Visualizer

An interactive web application that helps users understand graph traversal algorithms through real-time animations. Users can create custom graphs, visualize algorithm execution step by step, and control the animation speed.

## Features

- Interactive graph creation (add/remove nodes and edges)
- Depth-First Search (DFS) visualization
- Breadth-First Search (BFS) visualization
- Step-by-step node and edge animations
- Start, Pause, Resume, and Reset controls
- Adjustable visualization speed
- Responsive user interface
- Supports connected and disconnected graphs

## Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- DOM Manipulation

## Algorithms Implemented

- Breadth-First Search (BFS)
- Depth-First Search (DFS)

## Learning Outcomes

- Graph data structures
- Queue and Stack implementations
- DOM manipulation
- Algorithm visualization
- Responsive web development

## Improvements
- Dijkstras
